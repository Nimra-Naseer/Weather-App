# 🌤️ Weather Web App

A responsive Weather App built using HTML, Tailwind CSS, JavaScript, Axios, and WeatherAPI.

## ✨ Features

- Real-time weather data
- Temperature, humidity & wind info
- Weather icons
- Responsive design
- Stylish UI with Tailwind CSS
- Audio effect on search

## 🛠️ Technologies

- HTML5
- Tailwind CSS
- JavaScript
- Axios
- WeatherAPI

## 🚀 Run Project

1. Clone repository
2. Open `index.html`
3. Add your WeatherAPI key in `weather.js`
